Accept Pi Network cryptocurrency payments in your WooCommerce store.

== Description ==

**Pi Pay for WooCommerce** is the easiest way to accept Pi Network cryptocurrency payments on your WordPress store.

= Features =

* ✅ Accept Pi Network payments at checkout
* ✅ Easy setup - just add your Pi App ID and API Key
* ✅ Sandbox/Testnet mode for testing
* ✅ Automatic payment verification
* ✅ Order status auto-update after payment
* ✅ Live Pi exchange rate support
* ✅ Mobile friendly - works in Pi Browser

= How it works =

1. Customer adds products to cart
2. At checkout, customer selects "Pay with Pi"
3. Customer authenticates with Pi Network
4. Payment is created and verified automatically
5. Order is completed and customer redirected to thank you page

= Requirements =

* WordPress 5.0+

== Installation ==

1. Upload plugin to `/wp-content/plugins/pi-pay-woocommerce/`
2. Activate the plugin through 'Plugins' menu
3. Go to WooCommerce > Settings > Payments > Pi Network Payment
4. Enter your Pi App ID and API Key
5. Enable sandbox mode for testing
6. Save settings

== Frequently Asked Questions ==

= Where do I get Pi App ID and API Key? =

Register at [Pi Developer Portal](https://developers.minepi.com) and create a new app.

= Does it work with Pi Testnet? =

Yes! Enable Sandbox mode in settings to use Pi testnet for testing.

= What currencies are supported? =

Pi amount is calculated based on your WooCommerce currency and Pi exchange rate.

= Is it secure? =

Yes. All payments are verified server-side using Pi Network API before order completion.
